Mortgage Analysis Hub Documentation

Welcome to the Mortgage Hub. This system acts as a smart gateway, utilizing Advanced Machine Learning to evaluate Malaysian housing loan affordability based on authentic Bank Officer rules.

1. Core Evaluation Logic

The system evaluates applications using a robust Machine Learning algorithm (Random Forest) trained on a synthetic dataset representing realistic Malaysian financial profiles.

  • Debt Service Ratio (DSR): The system uses a dynamic DSR limit (up to 75% for experienced candidates) mapped against industry norms.
  • Net Disposable Income (NDI): An absolute buffer constraint. Urban locations typically require an NDI of RM1,500.
  • Credit Score / CCRIS: The system enforces a strict checking mechanism where scores below 650 will typically trigger high-risk rejections, mimicking actual bank operations.
2. Role-Based Access Control
  • Admin (Manager): Can access the full Dataset Explorer, trigger Model Retraining manually, manage other users, and access the "All Records" oversight panel.
  • User (Analyst): Focused purely on conducting new loan assessments, reviewing individual predictions, and exporting basic reports for their cases.
3. Export Monthly Report

The Export Monthly Report feature automatically packages the assessment history into a structured CSV file. Admins will receive a comprehensive dump of all applications, whereas standard users will only export the assessments they generated themselves.

It is recommended to generate this report on the 1st of every month to consolidate branch/team performance.
